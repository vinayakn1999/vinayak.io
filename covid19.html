Covid-19 Data Analysis Using SQL and Power BI

This project leverages SQL and Power BI to perform an in-depth analysis of the COVID-19 pandemic's global impact. By combining technical expertise and data storytelling, the project provides meaningful insights into vaccination progress, mortality rates, and comorbidities.

Objectives

  • Understand the trends in COVID-19 cases, deaths, and vaccinations.
  • Analyze the role of demographics and comorbidities in mortality rates.
  • Highlight vaccination milestones and their effect on controlling the pandemic.
  • Build interactive dashboards to visualize key findings.

Key Features

  • Exploratory Data Analysis (EDA): Using SQL, key questions like "Which country first reached a 25% vaccination milestone?" and "What is the rolling 7-day average of cases and deaths?" were answered.
  • Interactive Dashboards: Designed 5 dashboards in Power BI
      • 1.COVID-19 Analysis
      2.Demographic Analysis
      3.Vaccination Progress
      4. Mortality Insights
      5. Comorbidities Overview
  • Data Model: Created a comprehensive data model integrating multiple datasets for seamless querying and visualization.

    • Insights

  • Identified vaccination milestones and analyzed their influence on case trends.
  • Calculated and visualized mortality rates across continents and demographic groups.
  • Highlighted the role of comorbidities in COVID-19 outcomes.

    • Tools Used

  • SQL: For data cleaning, analysis, and answering critical business questions.
  • Power BI: For creating interactive visualizations and dashboards.
